INSTRUCCIONES PARA PUBLICAR EN GITHUB PAGES (SIN INSTALAR NADA)

1. Crea un repositorio público en GitHub llamado: sprint1-segmentacion
2. Entra al repositorio y haz clic en 'Add file' -> 'Upload files'.
3. Arrastra todo el contenido del ZIP (index.html, styles.css, assets/) a la zona de subida y confirma commit.
4. Ve a Settings -> Pages, selecciona Branch: main y folder: / (root). Guarda.
5. Espera 1-2 minutos y GitHub te dará la URL: https://tuusuario.github.io/sprint1-segmentacion/

SOBRE EL SITIO
- Título visible: Segmentación de Tarifas - Gerson Torrez
- El sitio usa Chart.js (CDN) para las visualizaciones interactivas.
- Reemplaza los gráficos o enlaces a Notebook/Repo modificando index.html.
